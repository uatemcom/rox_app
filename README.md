# Rutas y Opciones
Sistema de información que brinde datos a los diferentes usuarios y actores del sistema de transporte colectivo.

## Objetivo
Crear una red global del transporte público y privado de movimiento masico, interconectado a nivel nacional.

## Propuesta
Se contempla el desarrollo de un sistema que muestre a los usuarios del transporte colectivo las diferentes rutas y opciones disponibles para movilizarse de un punto a otro.

Se considera que se debe tener acceso a datos generados en tiempo real de los servicios que se encuentren en funcionamiento y las rutas que proveen, sus tarifas, horarios, tiempos de traslado y distancias por recorrer.
